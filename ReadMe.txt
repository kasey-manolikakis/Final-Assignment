# FinalAssignment

## Overview
This is a basic ASP.NET Core web application for creating a book list and adding books, editing them and deleting them. It uses Razor Pages / MVC for the frontend and Entity Framework Core for database operations.

---

## Features
- User registration and login (Identity)
- CRUD operations for Book
- Responsive design with Bootstrap

---

## Technologies
- ASP.NET Core 9.0
- Entity Framework Core
- SQL Server / SQLite (specify which you used)
- Razor Pages / MVC
- Bootstrap 5
